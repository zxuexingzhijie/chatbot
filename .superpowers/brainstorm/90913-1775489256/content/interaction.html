系统命令一览
游戏控制
continue — 剧情自动推进
undo — 回退上一步
retry — 重新生成本轮
save [名称] — 存档
load [名称] — 读档
信息查询
look — 查看环境
inventory — 查看背包
status — 角色状态
hint — 获取提示
help — 命令帮助
自由输入
直接用自然语言描述行动
例: "翻看吧台上的告示"
例: "偷偷溜到地下室门口"
例: "给格里姆讲个笑话"
意图解析流程
玩家输入
"和格里姆聊聊
地下室的事"
→
快速匹配?
关键词: 聊/说/问 → TALK
✓ 命中! 跳过LLM
→
ActionRequest
action: TALK
target: "bartender_grim"
detail: "地下室的事"
raw: "和格里姆聊..."
→
规则验证
✓ 格里姆在同一地点
✓ 格里姆非敌对状态
⚠ trust=-20, 态度冷淡
→
LLM叙事
基于trust=-20
生成冷淡/回避
的对话回应
⚡ 模糊输入降级路径
当关键词无法匹配时 → 调用LLM意图分类（few-shot prompt）→ 返回 action 类别 → 继续管道
例: "试着用魅力说服他" → LLM判定 → action: PERSUADE, target: bartender_grim, skill: charisma
