世界模型与数据结构

角色、地点、物品、关系、事件 — 以奇幻酒馆为例

核心数据模型 (dataclass / Pydantic)
## Character（角色）
Character:
  id: str                    # "bartender_grim"
  name: str                  # "格里姆老头"
  role: enum                 # NPC | PLAYER
  traits: list[str]          # ["暴躁", "善良内心", "酗酒"]
  stats: dict[str, int]      # {"hp": 100, "trust": 0}
  inventory: list[str]       # ["生锈钥匙", "酒杯"]
  location_id: str           # "tavern_bar"
  knowledge: set[str]        # {"知道地下室密道", "见过玩家偷酒"}

## Location（地点）
Location:
  id: str                    # "tavern_bar"
  name: str                  # "酒馆吧台"
  description: str           # 基础描述文本
  exits: dict[str, Exit]     # {"north": Exit("tavern_hall", locked=False)}
  items: list[str]           # ["破旧告示", "半杯麦酒"]
  npcs: list[str]            # ["bartender_grim"]

## Item（物品）
Item:
  id: str                    # "rusty_key"
  name: str                  # "生锈钥匙"
  description: str
  portable: bool             # 可否拾取
  usable_with: list[str]     # ["cellar_door"] 可交互目标

## Relationship（关系边）
Relationship:
  source: str                # "bartender_grim"
  target: str                # "player"
  type: enum                 # TRUST | FEAR | HOSTILITY | ALLIANCE
  value: int                 # -100 ~ 100

## Event（事件记录）
Event:
  id: str
  timestamp: int             # 游戏内回合数
  type: str                  # "dialogue" | "item_use" | "discovery"
  actor: str                 # 执行者
  description: str           # "玩家向酒保询问了地下室"
  consequences: list[str]    # ["bartender_trust -= 10"]
